# SQL-Based Customer Insights Analysis

## Project Overview
End-to-end SQL analysis of an e-commerce dataset to extract 
customer behaviour patterns, revenue trends, and business 
insights using MySQL.

## Business Questions Answered
1. Which product category generates the most revenue?
2. Who are the top 5 customers by total spending?
3. How is revenue trending month by month?
4. How can customers be segmented by purchase behaviour?
5. Which states generate the highest revenue?
6. How much revenue is lost due to order cancellations?

## Key Insights
- Electronics is the highest revenue-generating category
- June 2024 recorded peak monthly revenue
- Delhi is the top performing state by revenue
- 40% customer retention rate identified
- Gujarat and Rajasthan show highest cancellation rates

## Skills Demonstrated
- Multi-table JOINs (up to 4 tables)
- Aggregate functions (SUM, COUNT, COUNT DISTINCT)
- Date functions (DATE_FORMAT)
- CASE WHEN for customer segmentation
- Business-driven filtering with WHERE clauses
- GROUP BY and ORDER BY for reporting

## Tools Used
- MySQL 8.0
- MySQL Workbench

## Database Schema
```
customers → orders → order_items → products
```
